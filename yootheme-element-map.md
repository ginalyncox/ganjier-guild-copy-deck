# Ganjier Guild — YOOtheme Pro Element Map

> Last updated: April 28, 2026  
> Reference this when building or editing pages in YOOtheme Builder.

---

## Core structure rule

Every page follows: **Section → Row → Column → Element**

---

## Global element mapping

| Copy type | YOOtheme element | Notes |
|---|---|---|
| Eyebrow / kicker | Headline | Small heading style, uppercase if needed |
| Main headline | Headline | One H1 per page |
| Body paragraph | Text | Keep body copy in Text, not Headline |
| Primary / secondary CTA | Button | One Button element, multiple button items |
| 3-up benefits | Grid | Grid element with 3 items |
| Membership tier cards | Grid | Grid element with 2 items |
| FAQs | Accordion | One item per question |
| Portal side navigation | Nav | Grouped under Headline labels |
| Section divider | Divider | Use sparingly |
| Signup / contact form | Module Position or plugin embed | Keep forms in modules |

---

## Homepage

### Hero
Section → Row → 1 Column  
- Headline (eyebrow)  
- Headline (H1)  
- Text (subheadline)  
- Button (2 items: Join the Guild / Explore Member Benefits)  

**Style:** left aligned, width large, generous padding, Honey primary button, Mocha/transparent secondary button  
**Save as:** `GG - Hero - Standard`

### More than access
Section → Row → 1 Column  
- Headline  
- Text  

**Style:** muted/light warm background  
**Save as:** `GG - Section - Intro Text`

### Why members join
Section → Row → 1 Column  
- Headline  
- Grid (3 items: Education / Connection / Opportunity)  

**Style:** equal card heights, no oversized icons  
**Save as:** `GG - Grid - 3 Value Cards`

### Who the Guild is for
Section → Row → 2 Columns  
- Left: Headline + short intro Text  
- Right: Text with bullet list  

### Homepage CTA strip
Section → Row → 1 Column  
- Headline  
- Text  
- Button  

**Style:** distinct background, simpler than hero  
**Save as:** `GG - CTA Strip`

## Brand-to-element mapping

| Brand element | YOOtheme use |
|---|---|
| Covik Sans Bold | Headline elements, card titles, section headings |
| Covik Sans Bold ALL CAPS | Eyebrows, kicker labels, section preheads |
| Abadi MT Pro | Text elements, helper text, FAQs, descriptions |
| Honey | Primary CTA buttons, hover accents, active nav |
| Mint | Secondary accents, supporting highlights |
| Lavender | Tertiary accent only — not primary CTA |
| Mocha | Dark section backgrounds, footer, premium cards |
| Warning Yellow / Red | Alerts only — never decorative |

## Button styling

| Button | Style |
|---|---|
| Join the Guild | Honey fill, white text |
| Member Login | Mocha border + Mocha text, or Mocha fill + white text |
| Member Portal | Mocha fill + white text |
| Secondary/portal actions | Mint or neutral outline |

- Join the Guild  
- Activate Membership  
- Member Login
