# Ganjier Guild — Copy Deck & YOOtheme Implementation Docs

This repository contains the full website copy deck and YOOtheme Pro implementation reference for the Ganjier Guild site.

## Files

| File | Description |
|---|---|
| `copy-deck.md` | Full approved copy for all public pages — homepage, join, membership tiers, and reusable snippets |
| `yootheme-element-map.md` | Section-by-section guide mapping copy blocks to YOOtheme Pro elements (Section / Row / Column / Element) |
| `yootheme-implementation-checklist.md` | Page-by-page build checklist with exact element stacks, style notes, and final QA checklist |

## Usage

- Reference `copy-deck.md` when writing or editing page copy in YOOtheme Builder.
- Reference `yootheme-element-map.md` when deciding which element to use for each content block.
- Work through `yootheme-implementation-checklist.md` in order when building or auditing pages.

## Language rules

- Use **dues** not "membership fee"
- Use **Join the Guild** or **Activate Membership** not "Purchase Membership"
- Keep **Member Portal**, **My Directory Listing**, and **Memberships** (WooCommerce label) unchanged
